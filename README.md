blue-button-sandbox
===================

Client ONly Blue-Button Parser User Interface.

View at [github project page](http://amida-tech.github.io/blue-button-sandbox).

To run locally using [Jekyll](http://jekyllrb.com), use `$ bash serve.sh`.
